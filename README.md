# TurboBoost Switcher

**A macOS menu bar app to control Intel CPU Turbo Boost.**  
Made by **Dejan Njegić** — [dejan.njegic@gmail.com](mailto:dejan.njegic@gmail.com)

---

## Features

- **One-click toggle** — Enable or disable Turbo Boost from the menu bar
- **Remembers your last setting** — Stays on/off after restart
- **Real-time CPU monitoring** — Temperature, usage %, and frequency
- **Battery auto-switch** — Automatically disable Turbo on battery, enable on AC power
- **10 languages** — English, Serbian, German, French, Spanish, Italian, Portuguese, Russian, Chinese, Japanese
- **Launch at Login** — Starts automatically when you log in

## Requirements

- **Intel Mac** (x86_64) — does NOT work on Apple Silicon
- macOS 13 Ventura or newer

## Installation

Download the latest DMG from the [Releases](https://github.com/DejanVukovar/TurboBoostSwitcher/releases) page.

1. Open the DMG file
2. Double-click the PKG installer
3. Follow the wizard and enter your admin password when prompted
4. Launch **TurboBoost Switcher** from your Applications folder
